```
   - Bundle size: 500KB vs 1.2MB

3. Icons: Lucide vs Heroicons vs Phosphor
   - Stroke control, tree-shaking

4. Layout: CSS Grid vs Masonry Library
   - Browser support, flexibility

5. Animation: VueUse Motion vs GSAP
   - Use cases for each

6. State: Pinia vs Vuex vs Composables
   - API simplicity, TypeScript

7. Build: Vite vs Webpack
   - Cold start: 0.3s vs 8.5s

8. Data: YAML vs JSON vs Markdown
   - Readability, maintainability

9. Images: WebP + AVIF vs JPG/PNG
   - Compression: 60-75% smaller

10. Docker: Alpine vs Debian
    - Image size: 41MB vs 186MB
```

## üöÄ Quick Commands Reference

```bash
# Initialize project
npm create vite@latest recipe-app -- --template vue-ts
cd recipe-app

# Install dependencies
npm install vue@latest vue-router@4 pinia@latest \
  @vueuse/core @vueuse/motion lucide-vue-next \
  @yeger/vue-masonry-wall yaml

npm install -D vite-ssg @unhead/vue unocss @unocss/reset

# Development
npm run dev

# Build
npm run build

# Docker build
docker build -t recipe-app .

# Docker run
docker run -p 8080:8080 recipe-app

# Docker Compose
docker-compose up
```
