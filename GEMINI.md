# Project Overview

This directory contains a collection of recipes and cooking methods, organized by language (English and Spanish).

## Directory Structure

The directory is organized as follows:

-   `english/`: Contains recipes and methods in English.
    -   `recipes/`: Contains recipe files.
    -   `methods/`: Contains cooking method files.
-   `spanish/`: Contains recipes and methods in Spanish.
    -   `recipes/`: Contains recipe files in Spanish.
    -   `methods/`: Contains cooking method files in Spanish.

## Key Files

### English

*   `english/recipes/cilbir_recipe.md`: A recipe for Turkish Eggs with Yogurt (Çılbır).
*   `english/recipes/zucchini_ground_meat_stew.md`: A recipe for a hearty zucchini and ground meat stew.
*   `english/methods/poached_eggs_tips.md`: Tips and methods for making poached eggs.
*   `english/methods/roasted_tomato_sauce.md`: A method for making roasted tomato sauce.

### Spanish

*   `spanish/recipes/receta_cilbir.md`: La receta de Çılbır (Huevos a la Turca con Yogur).
*   `spanish/recipes/guiso_de_zapallo_italiano_y_carne_molida.md`: La receta de un guiso de zapallo italiano y carne molida.
*   `spanish/methods/consejos_huevos_escalfados.md`: Consejos y métodos para hacer huevos escalfados o pochados.
*   `spanish/methods/salsa_de_tomates_asados.md`: Un método para hacer salsa de tomates asados.

## Usage

This directory is intended to be a collection of recipes and cooking methods. You can add new recipes or methods by creating new markdown files in the appropriate language directory.
